/* How to run */

localhost에서 동작하도록 되어있고, 비밀번호는 1234로 설정하였다. 
 $conn = mysqli_connect("localhost","root","1234") ;  

1. 첨부된 MOVIE_DB.txt 파일 안의 데이터를 mysql 명령 프롬프트창에 복사 붙여 넣기 한다.
2. 첨부된 htdocs의 내용을 모두 실행 컴퓨터 xampp>htdocs안으로 이동시킨다. (폴더 전체를 덮어쓰기 하면 된다)
3. xampp control panel 에서 apache admin을 누른다. 
4. 맨 위에 Index of / 가 있고 밑에 php 파일들이 보인다. 그 중 login_screen.php를 클릭한다. 
5. login screen 이 뜬다. 가장 초기화면이 login screen이다. 
6. 스크린샷에 첨부된 대로 다양한 기능들이 동작하는지 확인할 수 있다.
